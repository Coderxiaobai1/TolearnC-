什么是EXIF？

　　EXIF(Exchangeable Image File format)是可交换图像文件的缩写，是专门为数码相机的照片设定的，可以记录数码照片的属性信息和拍摄数据。

　　EXIF可以附加于JPEG、TIFF、RIFF、RAW等文件之中，为其增加有关数码相机拍摄信息的内容和索引图或图像处理软件的版本信息。

佳能相机如何查询快门次数？

佳能相机没有在EXIF信息中记录快门次数，但40D以后的佳能相机，可以使用40DShutterCount 软件来查看快门次数。(点此下载 使用方法)
